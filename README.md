# 🚀 Futuristic 3D Developer Portfolio

A premium, immersive 3D portfolio website built with **Next.js 14**, **React Three Fiber**, **Framer Motion**, and **GSAP**. Features cinematic scroll animations, real-time 3D scenes, glassmorphism UI, and a magnetic custom cursor.

## ✨ Features

- **Real-time 3D** — Particle fields, floating geometry, and interactive avatar via React Three Fiber
- **Cinematic Animations** — Scroll-triggered reveals, parallax layers, and micro-interactions
- **Glassmorphism UI** — Frosted glass panels with metallic gradient borders
- **Custom Cursor** — Magnetic cursor with glow trail
- **Page Loader** — Cinematic loading screen with progress animation
- **Fully Responsive** — Mobile-first design with animated hamburger menu
- **Data-Driven** — All content lives in `data/*.json` files — update without touching UI code
- **SEO Optimized** — Open Graph, Twitter cards, semantic HTML, structured metadata

## 🛠 Tech Stack

| Technology | Purpose |
|---|---|
| Next.js 14 | Framework & SSR |
| React Three Fiber | 3D rendering |
| Framer Motion | Animations |
| GSAP | Advanced animations |
| Tailwind CSS | Styling |
| TypeScript | Type safety |

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view.

## 📁 Structure

```
portfolio-3d/
├── app/             # Next.js app directory
├── components/      # Reusable UI + 3D components
├── sections/        # Page sections (Hero, About, Skills, etc.)
├── data/            # JSON data files
└── public/          # Static assets
```

## 📝 Customization

Edit the JSON files in `data/` to update your content:
- `skills.json` — Your technology skills
- `projects.json` — Featured projects
- `experience.json` — Work experience
- `education.json` — Education history
- `testimonials.json` — Client testimonials
